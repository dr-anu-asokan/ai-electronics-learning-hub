# AI + Electronics Learning Hub

This repository is an open, hands-on learning hub that connects
**AI concepts to real hardware systems**.

It is designed for:
- students
- early engineers
- educators
who want to understand *where AI actually runs*.

---

## What This Repo Covers

- Arduino-based electronics + control
- Raspberry Pi system-level projects
- Jetson-based AI-hardware experiments
- Hardware constraints: power, memory, latency
- System thinking beyond code
